STM32CubeG4 Firmware Package V1.3.0 / 26-June-2020
  • HAL
    • Add support for STM32G491/4A1 part numbers
  • CMSIS Core
    • Upgrade to use new CMSIS Core version  V5.6
  • Middlewares
    • USB Device library upgrade to use new version of  V2.6.1
      • USB Core: Fix warning issue with Keil due to missing return value of setup API
      • Integration of three new USB device Class drivers:
        • USB CDC ECM Class driver
        • USB CDC RNDIS Microsoft Class driver
        • USB Billboard Class driver
    • USB-PD Core library upgrade to use V3.0.3 and G4 USBPD Device Driver upgrade to use V4.0.1, including :  
      • Maintenance corrections (including updates for Conformance testing)
      • Addition of a mechanism to avoid RX buffer overwrite in case of multiple RX
      • Management of Tx discard and Tx abort by UCPD IP
    •   STM32 OpenBootLoader library upgrade to use V2.0.0
      • Add support for new Bootloader I2C commands compatible with the Standard Bootloader I2C commands
